A PHP OUTPUT problem

用PHP的时候无意间发现一个问题

这个问题我去问了
Stack OverFllow 他们给出了比较详细的回答，这里我也做一下讲解
问题主要出在引用变量上面，在php中

$a = 7;
$d = ($c = &$a + 1); // $c和$d会出现不同的状况
echo $c; // 7 &运算符的优先权比+大
echo $d; // 8 这儿输出的是$c的值因为括号运算符内$c变量加上了1
```

这个问题存在于PHP语言本身的运算符优先权分配的问题，仅作新颖问题思考，实际使用中不能这样使用。
